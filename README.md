# Home Assistant Mobile

A specialized mobile client built with **React Native**, **Expo**, and **TypeScript** for managing and interacting with Home Assistant environments.

## 🚀 Features
- **Cross-Platform:** Built with React Native for seamless performance.
- **TypeScript Powered:** Ensures type safety and better developer experience.
- **Expo Integration:** Uses Expo for rapid development and easy testing.
- **Custom Android Configuration:** Includes a native Android directory for specialized hardware support or custom native modules.

## 🛠️ Tech Stack
- **Framework:** [Expo](https://expo.dev/) / React Native
- **Language:** TypeScript
- **State Management:** (Likely React Hooks / Context API based on structure)
- **Native Support:** Kotlin (Android)

## 📁 Project Structure
- `/src`: Contains the core application logic and UI components.
- `/android`: Native Android project files.
- `/assets`: Images, icons, and fonts used in the app.
- `App.tsx`: The main entry point of the application.
- `app.json` / `app.config.js`: Expo configuration files.

## 🏁 Getting Started

### Prerequisites
- Node.js
- Expo CLI (`npm install -g expo-cli`)
- A physical device or emulator with the Expo Go app.

### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/shawtygirlzzzz/home-assistant-mobile.git](https://github.com/shawtygirlzzzz/home-assistant-mobile.git)
2. Install dependencies:
   ```bash
   npm install
3. Start the development server:
   ```bash
   npx expo start
